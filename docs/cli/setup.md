# `firstclaw setup`

Initialize `~/.firstclaw/firstclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
firstclaw setup
firstclaw setup --workspace ~/.firstclaw/workspace
```

To run the wizard via setup:

```bash
firstclaw setup --wizard
```
