# `firstclaw tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
firstclaw tui
firstclaw tui --url ws://127.0.0.1:18789 --token <token>
firstclaw tui --session main --deliver
```
